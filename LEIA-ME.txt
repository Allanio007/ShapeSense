DESCRIÇÃO DO PROJETO
──────────────────────────────────────────────────────────────
ShapeSense é uma aplicação WebAR de apoio à consulta cirúrgica
que permite a simulação visual de próteses mamárias diretamente
no navegador do smartphone, em 3 estágios:

  Estágio 1 · Captura   — câmera traseira com guia de enquadramento
  Estágio 2 · Medição   — overlay RA via Canvas API detecta base,
                           projeção, distância intermamária e ptose
  Estágio 3 · Simulação — ajuste de volume, perfil, base e
                           superfície em tempo real + relatório PDF

TECNOLOGIAS UTILIZADAS
──────────────────────────────────────────────────────────────
  · HTML5 semântico
  · CSS3 (Custom Properties, Grid, Flexbox, animações)
  · JavaScript ES6+ (sem frameworks)
  · Canvas 2D API  — renderização RA e simulação de implantes
  · MediaDevices.getUserMedia — câmera traseira do dispositivo
  · window.print() — exportação de PDF local (sem servidor)

PRIVACIDADE
──────────────────────────────────────────────────────────────
  Processamento 100% local.
  Nenhuma imagem ou dado é transmitido a servidores.
  Nenhum cookie ou armazenamento persistente é utilizado.

CRÉDITOS E REFERÊNCIAS
──────────────────────────────────────────────────────────────

Este projeto não utiliza modelos 3D externos (.glb / .gltf).
A simulação é renderizada integralmente via Canvas 2D API
com algoritmo próprio de elipses com gradiente radial.
