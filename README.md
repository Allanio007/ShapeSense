# ShapeSense
Aplicativo para Simulação de Realidade Aumentada de ShapeSense

## Como executar

1. Abra o arquivo `index.html` em um navegador moderno.
2. Clique em **Iniciar câmera** e permita acesso à câmera.
3. Escolha forma, cor e tamanho para visualizar a sobreposição em tempo real.
